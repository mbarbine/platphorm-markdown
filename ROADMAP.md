# 🗺 MarkdownTree Roadmap

A living document outlining planned features, milestones, and the long-term vision for MarkdownTree.

---

## ✅ v1.0 — Foundation (Current)

- [x] Monaco-powered markdown editor
- [x] Interactive graph visualization with ReactFlow
- [x] Four view modes — Editor · Split · Graph · Preview
- [x] AI/model scaffolding with honest degraded state when no backend model provider is configured
- [x] Deterministic table-of-contents generation from real headings
- [x] Export to Markdown, HTML, JSON
- [x] Bounded URL-only share links with encoded content and no server storage claim
- [x] Dark / light theme
- [x] Document outline panel
- [x] Versioned REST API (`/api/v1`)
- [x] OpenAPI documentation at `/api/docs`
- [x] MCP configuration at `/.well-known/mcp.json`
- [x] SEO: Open Graph, Twitter Cards, JSON-LD, sitemap, RSS
- [x] Accessibility statement, privacy policy, terms of service

## 🚧 v1.1 — Polish & I18N

- [ ] 🌐 Full I18N support (English, Spanish, French, German, Japanese, Chinese)
- [ ] 🎨 Improved share cards with dynamic OG images
- [ ] 📱 Enhanced mobile editor experience
- [ ] ⌨️ Additional keyboard shortcuts
- [ ] 🔍 Full-text search within documents
- [ ] 📊 PNG graph export
- [ ] 🔄 Undo/redo history in editor
- [x] 💾 Browser-local draft persistence
